# 文档

## packge.json

```json
{
  //  程序入口
  // main 是commonJS模块 ，module 是 ES Module，
  // 被用户使用后，require() 或 import的时候 优先使用 module 的路径
  "main": "lib/index.js",
  "module": "es/index.js",
  //    指定ts的typing文件，方便用户使用的时候显示类型
  "typings": "index.d.ts",
  // 指定内部命令对应的可执行文件
  "bin": {
    "YOUR-SHELL-SCRIPT": "./bin/someTool.js"
  }
}
```
